# Wistia Upload Widget
This is an AngularJS component to upload videos to Wistia

## Requirements for development
- WebPack 
- NPM

## Requirements for running
- Web Server to host http-server

## How to develop it?
- Clone the project
- Run: `npm install`
- Run: `webpack --watch`
- Run: http-server -o

Demo: http://54.218.224.83:8080
